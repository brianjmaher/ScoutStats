This is a Python library to collect data from the FIRST Robotics Competition for use in making strategic decisions.

TO USE:
1) Run "main.py"
2) Input the event key, examples: "2016njfla", "2016scmb", "2016njbri", without quotes. The event code is the text that represents the event in The Blue Alliance URLs.
3) Open the newly-created .csv spreadsheets with Excel or another program of your choice.
